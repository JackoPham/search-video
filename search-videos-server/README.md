

## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```
## Generate

```bash
$ npm run generate
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# incremental rebuild (webpack)
$ npm run build:webpack
$ npm run start:webpack

# production mode
$ npm run production
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

- Authen restApi
- Get and create with graphql.
- Run ok with alias ==> debug, ts-node, node, webpack (import enties)
